# Bia da Candys - Landing Page 🌻

Landing page premium e mobile-first para a doceria gourmet **Bia da Candys**.

## 🚀 Funcionalidades

- **Design Premium**: Interface elegante com tema de girassóis e tons de rosa.
- **Formulário Inteligente**: Coleta dados de encomendas e os envia diretamente para uma planilha Google via integração com Google Forms.
- **WhatsApp Direct**: Após o envio do formulário, o usuário é redirecionado para o WhatsApp com uma mensagem pré-formatada contendo todos os detalhes do pedido.
- **Mobile-First**: Experiência otimizada para dispositivos móveis.

## 🛠️ Tecnologias

- **React** + **Vite**
- **Lucide React** (Ícones)
- **Vanilla CSS** (Design System customizado)
- **Google Forms API** (Submissão de dados)

## 📦 Como rodar o projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/Asproz21/Bia_Da_Candys.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
